## file structure
```
 1800-2024-DTC08/
├─ index.html

│  ├─ header.html
│  ├─ footer.html

├─ styles/
│  ├─ *.css
```
The sub-folders contain:
- **components**: Reusable HTML components.  
- **scripts**: All `*.js` files that define logic and user interaction.  
- **styles**: External stylesheets.

`skeleton.html` is a template html for all the pages.
